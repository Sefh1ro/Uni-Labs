# Lab-1-Image

## Завдання 1. Завантаження зображення
1. Підключіть модуль Image із бібліотеки PIL. Для цього напишіть та
запустіть на виконання команду:
 ```python
from PIL import Image
```
2. Відкрийте файл у поточному робочому каталозі. Для цього напишіть та
запустіть на виконання команду:

 ```python
python img = Image.open("img1.jpg") 
# Виводимо зображення на екран: 
img.show()
```

3. Відкрийте цей файл, розташований у поточному робочому каталозі, в
бінарний режим. Для цього напишіть та запустіть на виконання команду:
 ```python
img = Image.open("Tulips.jpg")
img = img.convert("L")
```
а в раніше, ніж Release 7.2.0.dev0 Pillow виконуємо команду
 ```python
img = Image.open("img1.jpg", "rb")
# Виводимо зображення на екран:
img.show()
```
4. Замість вказівки шляху до файлу можна передати файловий об'єкт,
відкритий у бінарному режимі. Для цього напишіть та запустіть на
виконання команди:
 ```python
# Відкриваємо файл у бінарному режимі
f = open("img2.gif", "rb")
# Передаємо об'єкт файлу
img = Image.open(f)
# Виводимо передане зображення на екран:
img.show() f.close()
# Закриваємо файл
```
5. Завантажте зображення з рядка, використовуючи модуль BytesIO
(більше ранньої версії StringIO), попередньо відкривши файл у
бінарному режимі:
```python
Import io
from PIL import Image
f = open("img2.gif", "rb")read()
img = Image.open(io. BytesIO(f))
img = img.convert("L")
# Виводимо зображення на екран:
img.show()
```
